# test-video

FindEye 데모·검증용 테스트 CCTV 영상 모음. 실제 CCTV와 유사한 장면을 생성형 AI로 제작해, 상의·하의 밝기 조건별 검색 동작을 확인하는 데 사용한다.

## 영상 목록

| 파일 | 장소 | 해상도 | 길이 |
|------|------|--------|------|
| `park_cctv.mp4` | 공원 | 1920×1080 | 29초 |
| `market-cctv.mov` | 시장 | 1920×1080 | 30초 |
| `school-cctv.mp4` | 학교 | 1280×720 | 10초 |

## 메타데이터 매칭

앱에 영상을 업로드하면 파일명을 기준으로 `data/video_metadata.csv`에서 장소·시작 시간을 자동으로 불러온다. 파일명이 CSV와 정확히 일치해야 매칭된다.

- `market-cctv.mov`, `park_cctv.mp4` → CSV에 등록되어 있어 장소·시간이 자동 입력된다.
- `school-cctv.mp4` → CSV에 미등록 상태이므로 직접 입력하거나 CSV에 추가한다.

CSV에 추가할 경우 예시:

```csv
school-cctv.mp4,학교,2026-06-24 21:43:00
```

## 사용 방법

1. `streamlit run app/streamlit_app.py`로 앱 실행
2. 위 영상 중 하나를 업로드
3. 상의/하의 밝기 조건 선택 후 검색
4. 조건에 맞는 후보가 장소·탐지 시각·신뢰도와 함께 출력됨
